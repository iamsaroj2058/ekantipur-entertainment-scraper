# Task 1 – Entertainment News Scraper


The goal of this task is to extract the top 5 entertainment news articles from
ekantipur.com using browser automation with Playwright and Python.



## Objective

- Navigate to the Entertainment (मनोरञ्जन) section of ekantipur.com
- Extract the top 5 news articles
- Capture structured data for each article:
  - Title
  - Image URL
  - Category
  - Author (if available)

The extracted data is saved in a JSON file for further processing and validation.



## Tech Stack

- Python 3.11+
- Playwright (Browser Automation)
- uv (Python Package Manager)
